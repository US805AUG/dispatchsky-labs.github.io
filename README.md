## Purpose

Created in 2025 to host an `app-ads.txt` file for an experimental Google AdMob integration.

## Outcome

The project moved away from AdMob because the Flight Dispatch Study user base was too small for advertising to be worthwhile.

This repository is retained as a historical reference in case AdMob is reconsidered in the future.
