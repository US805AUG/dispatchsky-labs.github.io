# Sadiom.com

A modern, responsive website for Sadiom - showcasing innovation and excellence.

## Overview

This is the official website for sadiom.com, built with modern web technologies and best practices. The site features a clean, professional design with smooth animations and full mobile responsiveness.

## Features

- 🎨 Modern, beautiful UI with gradient accents
- 📱 Fully responsive design for all devices
- ⚡ Smooth scrolling and animations
- 🚀 Fast loading and optimized performance
- 💼 Sections for Home, About, Services, and Contact
- 📧 Interactive contact form
- 🎯 SEO-friendly structure

## Technologies Used

- HTML5
- CSS3 (with CSS variables and modern layouts)
- JavaScript (Vanilla JS with ES6+)
- Google Fonts (Inter)

## Project Structure

```
/
├── index.html          # Main HTML file
├── styles.css          # Stylesheet
├── main.js            # JavaScript functionality
├── CNAME              # Custom domain configuration
├── app-ads.txt        # AdMob configuration
└── README.md          # This file
```

## Custom Domain

This site is configured to use the custom domain `sadiom.com`. The CNAME file ensures proper GitHub Pages configuration.

## Sections

1. **Hero**: Eye-catching landing section with call-to-action buttons
2. **About**: Highlights key values - Innovation, Performance, and Precision
3. **Services**: Showcases four main service offerings
4. **Contact**: Interactive contact form for inquiries
5. **Footer**: Quick links and social media connections

## Development

To work on this project locally:

1. Clone the repository
2. Open `index.html` in your browser
3. Make changes to HTML, CSS, or JS files
4. Refresh to see updates

## Deployment

This site is automatically deployed via GitHub Pages. Any push to the main branch will update the live site at sadiom.com.
